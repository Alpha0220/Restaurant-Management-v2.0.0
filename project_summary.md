# สรุปรายละเอียดโปรเจกต์ Restaurant Management System

## ภาพรวม (Overview)
โปรเจกต์นี้เป็นเว็บแอปพลิเคชันสำหรับ **ระบบจัดการร้านอาหาร (Restaurant Management System)** ที่พัฒนาด้วย **Next.js 16** และ **React 19** โดยใช้ **Google Sheets** เป็นฐานข้อมูลแบบ Real-time ขนาดเล็ก สำหรับจัดการสต็อกสินค้า, ข้อมูลเมนู, และบันทึกยอดขาย

## เทคโนโลยีที่ใช้ (Technology Stack)
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Database**: Google Sheets (ใช้งานผ่าน `google-spreadsheet` และ `google-auth-library`)
- **Authentication**: ระบบ Login อย่างง่ายสำหรับ Admin (เก็บค่าใน Environment Variable)

## ฟีเจอร์หลักและการเชื่อมต่อ Google Sheets
ระบบใช้ Google Sheets ในการจัดเก็บข้อมูลหลัก 3 ส่วน ได้แก่:
1. **Stock (สต็อกสินค้า)**: บันทึกประวัติการนำเข้าสินค้า (ชื่อ, จำนวนมราคา, ผู้บันทึก, วันที่)
2. **Menu (เมนูอาหาร)**: รายการเมนูสำหรับขาย (ชื่อ, ราคา, ส่วนผสม)
3. **Orders (คำสั่งซื้อ)**: ประวัติการขาย (รายการสินค้า, ราคารวม, ต้นทุนรวม, วันที่)

## รายละเอียดแต่ละหน้า (Page Breakdown)

### 1. Login (`/login`)
- **หน้าที่**: จุดเข้าใช้งานสำหรับผู้ดูแลระบบ (Admin)
- **ฟีเจอร์**:
  - ยืนยันตัวตนด้วย Username/Password
  - ตรวจสอบความถูกต้องกับข้อมูลใน Server-side Environment Variables
  - จัดการ Session อย่างง่ายด้วย Cookie

### 2. Dashboard (`/dashboard`)
- **หน้าที่**: แสดงข้อมูลทางธุรกิจและสรุปยอดขาย
- **ฟีเจอร์**:
  - แสดงตัวเลขทางการเงิน: ยอดขายรวม (Total Sales), ต้นทุนสินค้าขายรวม (COGS), กำไรขั้นต้น (Gross Profit), กำไรสุทธิ (Net Profit)
  - กรองข้อมูลได้ตาม: วัน (Day), เดือน (Month), ปี (Year) หรือช่วงวันที่กำหนดเอง (Custom Date Range)
  - แสดงกราฟหรือข้อมูลสรุปยอดขาย

### 3. POS (Point of Sale) (`/pos`)
- **หน้าที่**: หน้าขายหน้าร้านสำหรับพนักงาน
- **ฟีเจอร์**:
  - **Menu Grid**: แสดงรายการเมนูพร้อมราคา
  - **Cart System**: ตะกร้าสินค้า สามารถเพิ่มรายการ, ปรับจำนวนสินค้า และลบรายการได้ มี Modal กรอกจำนวนสินค้าเพื่อความแม่นยำ
  - **Checkout**: คำนวณยอดเงินรวมและบันทึกคำสั่งซื้อลงฐานข้อมูล
  - **Responsive Design**: 
    - **Desktop**: แบ่งหน้าจอ ซ้ายสั่งอาหาร ขวาสรุปรายการ
    - **Mobile**: แสดงเมนูเต็มจอ พร้อมปุ่มลอยเพื่อเปิดดูตะกร้าสินค้า (Bottom Sheet/Modal)

### 4. Menu Management (`/menu`)
- **หน้าที่**: จัดการรายการสินค้าที่ขาย
- **ฟีเจอร์**:
  - เพิ่มเมนูใหม่ (ระบุชื่อ, ราคา, วัตถุดิบ)
  - แก้ไขข้อมูลเมนูเดิม
  - ลบเมนู

### 5. Stock Management (`/stock`)
- **หน้าที่**: ติดตามและจัดการสต็อกสินค้า
- **ฟีเจอร์**:
  - **Add Stock**: เพิ่มรายการสต็อกทีละรายการ
  - **Bulk Import**: รองรับการเพิ่มข้อมูลหลายรายการพร้อมกัน
  - **History**: ดูประวัติการเพิ่มสต็อกเรียงตามเวลา

## การรองรับหน้าจอ (Responsive - Mobile vs PC)
แอปพลิเคชันถูกออกแบบมาให้รองรับทุกอุปกรณ์ (Fully Responsive) ด้วยแนวคิด "Mobile-First"
- **Mobile (มือถือ)**: 
  - การจัดวางเป็นคอลัมน์เดียว (Single-column layout)
  - การนำทางและส่วนที่ซับซ้อน (เช่น ตะกร้าสินค้า) จะซ่อนใน Modal หรือใช้ปุ่ม Toggle เพื่อประหยัดพื้นที่หน้าจอ
  - ปุ่มและช่องกรอกข้อมูลขนาดใหญ่ รองรับการกดด้วยนิ้วสัมผัส
- **PC/Tablet (คอมพิวเตอร์/แท็บเล็ต)**:
  - ใช้ประโยชน์จากหน้าจอแนวกว้าง แสดงผลแบบหลายคอลัมน์ (เช่น เมนูในหน้า POS แสดง 4 รายการต่อแถว)
  - จัดวาง Layout แบบ Side-by-side เพื่อการทำงานที่คล่องตัว (เช่น เมนูอยู่คู่กับตะกร้าสินค้า)
